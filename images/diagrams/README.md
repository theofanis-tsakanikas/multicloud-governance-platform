# Διαγράμματα αρχιτεκτονικής — σχεδιασμένα, όχι παραγόμενα από AI εικόνα

Δέκα διαγράμματα, ένα ανά prompt του [`../prompts/`](../prompts/). Κάθε ένα υπάρχει σε **SVG**
(διανυσματικό, το κείμενο είναι κείμενο) και σε **PNG** (3200×1800, για να το κολλήσεις όπου θες).

Αυτά **δεν** βγήκαν από image generator. Είναι σχεδιασμένα με κώδικα — γι' αυτό κάθε label είναι
ακριβώς αυτό που λέει ο κώδικας και η υποδομή, και όχι «Amazom RSD Proxi».

| # | Αρχείο | Τι δείχνει |
|---|---|---|
| **10** | `10-the-whole-platform` | 🎬 **Άνοιγμα** — ένα JSON contract → 3 clouds → 2 μηχανές |
| 01 | `01-aws-public-connection` | AWS **δημόσια** — Databricks → RDS, απευθείας |
| 08 | `08-azure-and-gcp-public` | Azure + GCP **δημόσια** — δύο κόμβοι, ένα άλμα |
| **05** | `05-the-transit-hub-idea` | 🥇 **Η ιδέα** — *γιατί* το NCC δεν περνάει σε άλλο cloud |
| 02 | `02-aws-private-connection` | AWS **ιδιωτική** — NCC → PrivateLink → pgbouncer → RDS |
| 06 | `06-azure-private-connection` | Azure **ιδιωτική** — transit hub + IPsec + private endpoint |
| 07 | `07-gcp-private-connection` | GCP **ιδιωτική** — transit hub + το private VIP της Google |
| **09** | `09-three-clouds-private-hero` | 🥇 **ΤΟ HERO** — 3 clouds, 3 ιδιωτικές διαδρομές, 1 εικόνα |
| 03 | `03-public-vs-private-side-by-side` | Δημόσιο vs ιδιωτικό — «ένας διακόπτης, όχι ξαναγράψιμο» |
| 04 | `04-zero-copy-snowflake` | 🎬 **Κλείσιμο** — ένα gold layer, δύο μηχανές, μηδέν αντίγραφα |

**Αν έχεις χρόνο για τρία:** **10** (τι είναι) → **05** (γιατί έτσι) → **09** (τι χτίστηκε).

---

## Ξαναφτιάξ' τα

```bash
python3 images/diagrams/render.py           # SVG + PNG (χρειάζεται Chrome για το PNG)
python3 images/diagrams/render.py --svg     # μόνο SVG — καμία εξάρτηση
python3 images/diagrams/render.py --only 09 # ένα μόνο
```

Ο [`render.py`](render.py) έχει **ένα** design system στην κορυφή (παλέτα, κόμβοι, βέλη, λουκέτα,
tunnels, cloud boundary) και δέκα δηλωτικούς ορισμούς από κάτω. Αλλάζεις ένα όνομα σε ένα σημείο,
ξανατρέχεις, και τα δέκα μένουν συνεπή μεταξύ τους.

Το PNG παράγεται με headless Chrome στα 2× (1600×900 → 3200×1800).

## Οι συμβάσεις που κουβαλάνε νόημα

Δεν είναι διακοσμητικές — κάθε μία λέει κάτι:

- 🔒 **πράσινο λουκέτο** = ιδιωτικό hop. 🔓 **πορτοκαλί ανοιχτό** = δημόσιο.
- ☁️ **σύννεφο internet** = η κίνηση βγαίνει στο δημόσιο δίκτυο. **Δεν υπάρχει σε κανένα από τα
  ιδιωτικά διαγράμματα** — η απουσία του *είναι* το επιχείρημα.
- **cloud boundary** (διακεκομμένη) = το όριο μεταξύ clouds. Στο `09` τη διασχίζουν **μόνο τα δύο
  IPsec tunnels**, και δεν αγγίζει καν τη λωρίδα του AWS — γιατί η διαδρομή του AWS δεν φεύγει
  ποτέ από το AWS.

## Ακρίβεια

Κάθε όνομα, CIDR, θύρα και ARN έρχεται από τους πίνακες «🔎 Επαληθευμένα» των prompts — που με τη
σειρά τους διαβάστηκαν από τον κώδικα και τη ζωντανή υποδομή. Ειδικά:

`199.36.153.8/30` (`private`, **όχι** το `restricted` `.4/30`) · `10.40.0.0/16` (RDS) ·
`10.10.0.0/16` (Azure) · `10.11.0.0/16` (GCP) · NLB `:5432` / `:1433` / `:443` · HAProxy container
`:8443` · `arn:aws:iam::565502421330:role/private-connectivity-role-eu-central-1`
